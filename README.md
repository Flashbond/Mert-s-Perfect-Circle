MERT'S PERFECT CIRCLE



Create perfectly circular roads with adjustable diameter and live preview.





FEATURES

* Live preview before placement
* Precise diameter control (Ctrl + Mouse Wheel)
* Tool panel integration
* Cost preview
* Supports snapping and elevation
* Displays outer and inner diameter (meters and units)





USAGE

1. Select a road
2. Activate Perfect Circle
3. Adjust diameter
4. Click to place





COMPATIBILITY

Currently supports standard road types only. Some road types (bridges, dirt roads, pedestrian paths, alleys) are not supported to maintain geometric consistency.





INSTALLATION

1. Download latest release
2. Extract to:
%AppData%\\LocalLow\\Colossal Order\\Cities Skylines II\\Mods
3. Launch the game
